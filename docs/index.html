This project allows the use of the Pyro Network Control Protocol on the Arduino platform
PNCP for Arduino

PNCP is the Pyro Network Control Protocol. This is a open source networking protocol developed by Leon S. Searl for use with hobbiest to professional firework firing systems.

Warning:

This is to be used at your own risk, we are not responsible for any harm to you or your property. This is currently in it's alpha stage and should not be used in the field.
